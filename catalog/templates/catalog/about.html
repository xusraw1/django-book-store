Привет, меня зовут Хусрав Махмаджанов

Мне 21 год, и я живу в прекрасном городе Карши, Узбекистан. В настоящее время я учусь на 4 курсе Каршинском филиале Ташкентского университета информационных технологий (ТАТУ) по специальности "Программный инжиниринг". Этот проект — мой дипломный, и он является важным шагом в моей профессиональной карьере.

Идея создания этого онлайн-магазина книг появилась у меня давно, но реализовать её было непросто. Однако сейчас, благодаря накопленному опыту и знаниям, я смог воплотить её в жизнь. Этот проект стал для меня не только вызовом, но и возможностью применить свои навыки на практике.

Цель проекта

Этот проект — не просто часть моей дипломной работы. Это результат моей страсти к программированию и желания создать что-то полезное для людей. Я верю, что книги — это источник знаний и вдохновения, и мой магазин поможет каждому найти именно ту книгу, которая изменит его жизнь.

Технологии и ресурсы

Для создания этого проекта я использовал следующие технологии и ресурсы:

  • Django — фреймворк для быстрой разработки веб-приложений на Python. Официальный сайт
  • Bootstrap — библиотека для создания адаптивных и современных интерфейсов. Официальный сайт
  • Django Allauth — библиотека для аутентификации и регистрации пользователей. Документация
  • SQLite — встроенная база данных для разработки. Официальный сайт
  • Git и GitHub — система контроля версий и платформа для хранения кода. Официальный сайт Git, GitHub
  • Docker — инструмент для контейнеризации приложений. Официальный сайт
  • Stack Overflow — платформа для поиска решений проблем. Перейти на сайт
  • MDN Web Docs — документация по HTML, CSS и JavaScript. Перейти на сайт

На создание этого проекта ушло около 2 месяцев, включая изучение новых технологий, разработку и тестирование.

Почему этот проект важен для меня?

Создание этого магазина стало для меня настоящим приключением. Я столкнулся с множеством вызовов, но каждый из них помог мне стать лучше как разработчику. Этот проект — не только шаг к получению диплома, но и важный этап моего профессионального роста.
